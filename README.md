# Yuri & David's Wedding Invitation

An interactive wedding invitation website featuring an envelope that opens to reveal the invitation.

## Deployment

This site can be deployed to **GitHub Pages**:

1. Go to your repository: https://github.com/githandle1/yuri
2. Click on **Settings**
3. Scroll down to **Pages**
4. Under "Source", select **Deploy from a branch**
5. Choose **main** branch and **/ (root)** folder
6. Click **Save**

Your site will be live at: `https://githandle1.github.io/yuri/`

## Local Development

Run locally with:

```bash
python3 -m http.server 9999
```

Then visit: http://localhost:9999
